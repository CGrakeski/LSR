# 高阶函数：匿名函数基本语法：
```
// 常规形式
do |arguments| {
    statements
};

// 无参数时简写
do {
    statements
};

// 简写形式
|argument| expression
```
### 使用示例：
```
// 简写形式定义函数
var float_add: callable[float, [float,float]] = |a,b| => a+b

// 常规形式定义函数
var int_add: callable[int,[int,int]] =
do |a,b| {
    return a+b
}
```

## 装饰器

---
### 核心语法：
```
@function_name  // 装饰器函数
func name(arguments){
    statements
}
```
### 使用示例：
```
@pure  // 标记函数为纯函数（无副作用）
func add(a,b){
    return a+b
}
```
