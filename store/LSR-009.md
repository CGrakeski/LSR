## 摘要

本LSR提出在Lamina语法内实现续行功能。

## 技术规范

### 反斜杠续行

当某行的行末是反斜杠`\`时，忽略这个反斜杠，去除行尾的换行符（`\n`）。如下面的代码：
```
var a = 1 \
+ 1;  // 注意空格！
```
应该被解释成
```
var a = 1 + 1;  // 注意空格！
```

### 不完整括号续行

当某一行中的括号不能配对时，应去除行尾的换行符`\n`。如下面的代码：
```
var a = (1 +
         1);
```
应该被解释成
```
var a = (1 + 1);
```
